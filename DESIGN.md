---

# 6. Logo Usage Rules

## Minimum Clear Space
Maintain clear space equal to the height of the “V” around the logo.

## Minimum Size
Digital:
- 120px width minimum

Print:
- 25mm width minimum

## Incorrect Usage
Do NOT:
- stretch logo
- rotate logo
- add glow effects
- apply gradients
- use outdated typography
- add drop shadows
- place on low-contrast backgrounds
