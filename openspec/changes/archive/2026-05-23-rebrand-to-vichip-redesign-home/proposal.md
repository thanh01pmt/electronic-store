## Why

The platform needs a complete brand transition from "Circuit Parts" / "ELE Store" to **VICHIP Electronics** in accordance with the new brand design system (`DESIGN.md` and `brand-guideline.png`). Additionally, the homepage needs to be redesigned to project a premium industrial-tech aesthetic and implement key discovery features inspired by `thegioiic_research_report.md` to enhance the user search and onboarding experience.

## What Changes

- Rebrand the site from "ELE Store" to "VICHIP Electronics" across all components, layout, document titles, logos, colors, and email templates.
- Update the color scheme to VICHIP green (`#00A651`) and dark green (`#006B3F`) as primary accents, combined with tech black and industrial grays.
- Implement the "Exo 2" typeface for headlines and title elements, and ensure "Inter" is utilized for body text and technical tables.
- Redesign the homepage layout to match the design mockups in `brand-guideline.png` (including a modern Dark/Light Hybrid hero banner, value-proposition metrics, a dynamic visual category explorer grid, and quick links).
- Add direct integration points for the BOM Tool (Excel upload) and dynamic RFQ triggers on the homepage.

## Capabilities

### New Capabilities
- `brand-identity`: System-wide VICHIP brand elements, theme configuration, color tokens, and static assets.
- `homepage-discovery`: A redesigned landing page with quick component category navigation, industrial value proposition cards, and quick-access BOM/RFQ triggers.
