## ADDED Requirements

### Requirement: VICHIP Color Scheme
The system SHALL display the brand colors of VICHIP Electronics on all user-facing web pages, specifically employing VICHIP Green (`#00A651`) for primary actions, badges, and accents, and Tech Black (`#1E1E1E`) for primary dark sections/layouts.

#### Scenario: Visual Accent Matching
- **WHEN** user views the website header or buttons
- **THEN** they see elements styled with VICHIP Green (#00A651) and Tech Black (#1E1E1E)

### Requirement: Typography Enforcement
The system SHALL use the "Exo 2" font family for all section headings, titles, and main call-outs, and the "Inter" font family for body text, tables, forms, and product descriptions.

#### Scenario: Heading Style Verification
- **WHEN** user views page headings
- **THEN** they see headings rendered using Exo 2 font-family and body text rendered using Inter font-family

### Requirement: Header and Footer Rebranding
The system SHALL display the VICHIP Electronics wordmark and logo in the header and footer, replacing all legacy references to "ELE Store" or "Circuit Parts".

#### Scenario: Brand Verification in Footer
- **WHEN** user scrolls to the footer of the page
- **THEN** they see "VICHIP Electronics" and copyright notices referencing "VICHIP Electronics"
