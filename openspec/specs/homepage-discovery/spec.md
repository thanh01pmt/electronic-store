## Requirements
### Requirement: Dark Hybrid Hero Banner
The homepage SHALL include a dark hybrid hero banner with an industrial-tech aesthetic featuring a glowing MCU/PCB image, a main headline "Reliable Components. Powering Innovation." (with "Innovation." highlighted in VICHIP green), and two primary call-to-action buttons: "REQUEST RFQ" and "VIEW PRODUCT LINES".

#### Scenario: Hero Banner CTA Interaction
- **WHEN** user clicks on "REQUEST RFQ" button
- **THEN** they are redirected or presented with the RFQ page/form

### Requirement: Key Value Propositions
The system SHALL display four key industrial value propositions with vector icons immediately beneath the hero banner: "100% Authentic Products", "50K+ Products in Stock", "Fast Delivery", and "Technical Support".

#### Scenario: Value Cards Displayed
- **WHEN** user scrolls past the hero banner
- **THEN** they see the four columns of value propositions with descriptions and corresponding line icons

### Requirement: Product Category Explorer Grid
The homepage SHALL present an interactive grid of nine main product categories with geometric line-based icons: IC, MCU, Power, Sensor, Connector, Relay, Industrial Automation, Semiconductor, and Embedded. Clicking on a category redirects the user to the corresponding filtered parts/products view.

#### Scenario: Category Grid Redirection
- **WHEN** user clicks on the "IC" category icon
- **THEN** they are redirected to the search results page filtered for IC components
